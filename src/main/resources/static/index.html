Welcome to the Natural Language Processing Article Library.
The NLP library uses the Text Classification is MeaningClouds solution for automated document classification. It assigns one or more categories to a text, using standard domain-specific taxonomies (e.g., IPTC. IAB, ICD-10) or
user-defined categories. The algorithm combines statistical document classification with rule-based filtering, which allows
to obtain a high degree of accuracy and flexibility in a wide range of environments.
